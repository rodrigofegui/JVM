5 ## Ferramentas
6 
7 Sendo um código desenvolvido em C/C++ o analisador estático é o [CppCheck](https://github.com/danmar/cppcheck) na versão 1.89 (a mais recente até o momento), podendo ser instalado, em Linux, a partir dos comandos:
8 
9 ```
10 git clone git://github.com/danmar/cppcheck.git
11 cd cppcheck/
12 # Substituir pela mais recente
13 git checkout 1.89
14 make MATCHCOMPILER=yes FILESDIR=/usr/share/cppcheck HAVE_RULES=yes CXXFLAGS="-O2 -DNDEBUG -Wall -Wno-sign-compare -Wno-unused-function"
15 sudo FILESDIR=/usr/share/cppcheck make install
16 # Testando se foi instalado
17 cppcheck --version
18 ```
19 
20 Enquanto que para o analisador dinâmico, é utilizado o [Valgrind](http://www.valgrind.org/) na versão 3.15.0 (a mais recente até o momento), podendo ser instalado, em Linux, a partir dos comandos:
21 
22 ```
23 wget https://sourceware.org/pub/valgrind/valgrind-3.15.0.tar.bz2
24 tar -jxvf valgrind-3.15.0.tar.bz2
25 cd valgrind-3.15.0/
26 ./configure
27 make
28 sudo make install
29 # Testando se foi instalado
30 valgrind --version
31 ```
